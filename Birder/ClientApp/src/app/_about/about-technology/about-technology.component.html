The technology stack:
  • ASP.NET Core and C# server-side API1
  • Angular and TypeScript client-side application2 with Bootstrap layout and styling
  • Hosted on Azure with Docker

1 As of 26/09/2019, ASP.NET Core version 3.0 is in use. It previously used .NET Core 2.2.
2 As of 12/09/2019, Angular version 8 is in use. Angular versions 6 and 7 were previously used.
