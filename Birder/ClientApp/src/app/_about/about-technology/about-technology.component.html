The technology stack
  • ASP.NET Core and C# server-side API1
  • Angular and TypeScript client-side application2 with Bootstrap layout and styling
  • Hosted on Azure with Docker

1 As of 05/01/2020, ASP.NET Core version 3.1 is in use. It previously used .NET Core 2.2 and 3.0.
2 As of 04/03/2020, Angular version 8 is in use. Angular versions 6, 7 and 8 were previously used.
The source code is available on GitHub for downloading, forking, or contributing
