Birder is built with:

  • ASP.NET Core and C# server-side API1
  • Angular and TypeScript for client-side single-page application2
  • Bootstrap layout and styling

1 As of 12/09/2019, this app uses .NET Core version 2.2.
2 As of 12/09/2019, the app is built in Angular version 8. It was previously built with Angular versions 6 and 7.
