Birder is built with:

  • ASP.NET Core and C# server-side API1
  • Angular and TypeScript for the client-side single-page application2
  • Bootstrap layout and styling

1 As of 12/09/2019, .NET Core version 2.2 is in use.
2 As of 12/09/2019, Angular version 8 is in use. Angular versions 6 and 7 were previously used.
