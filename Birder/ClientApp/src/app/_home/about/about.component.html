The technology stack
  • ASP.NET 5 and C# server-side API1
  • Angular and TypeScript client-side application2 with Bootstrap layout and styling
  • Hosted on Azure with Docker

1 Since 13th November 2020, this software uses .NET 5. ASP.NET Core versions 2.2, 3.0 and 3.1 were previously used.
2 Since 15th May 2021, this website uses Angular 12. Angular versions 6, 7, 8, 9, 10, and 11 were previously used.
The source code is available on GitHub for downloading, forking, or contributing
